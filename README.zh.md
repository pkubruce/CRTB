# CRTB

English | [中文](README.zh.md)

基于扩展版中国结核病复发（CRTB）模型的 Shiny 交互式平台。

本仓库包含扩展版中国结核病复发（CRTB）模型分析的代码和数据。项目使用 `renv` 包（版本 1.0.11）进行环境管理。完整的 R 环境（包括所有依赖包）已预装在 `renv/library/` 文件夹中，解压后即可直接运行代码，无需额外安装。

## 数据

- 所有结核病负担数据均来源于世界卫生组织（WHO）《2025年全球结核病报告》，存放在 `GTB2025/` 文件夹中。

## 代码结构

分析流程如下：

- **`R/`** – 包含分析所用的核心工具函数。
- **`01_fitted.R`** – 对模型进行校准，匹配 WHO 的流行病学目标数据。
- **`02_simulation.R`** – 运行基线模拟，预测结核病负担。
- **`03_Intervention.R`** – 模拟四种干预场景（高危/低危复发人群疫苗接种、LTBI 人群 TPT 治疗、联合干预）。
- **`04_Manuscript.R`** – 生成论文中呈现的图表和汇总统计结果。
